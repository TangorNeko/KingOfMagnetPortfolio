# **ブルーム**
川瀬式ブルームを実装。  
通常シーンをレンダリング後、輝度が高いピクセルを抽出し、  
ブラーとダウンサンプリングをかけながら複数枚のテクスチャを作成し、  
複数枚のテクスチャの平均を取って加算合成しています。

# **シャドウ**
今回は平行光源によるデプスシャドウを実装しています。  
平行光源のため、深度値をカメラからの距離ではなく、  
カメラ空間のXY平面からの最短距離を計算して使用しています。
